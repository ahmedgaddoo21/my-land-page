## 📌 Project Overview
The goal of this project is to create a professional landing page that is:  
- Fully responsive across all devices (Mobile, Tablet, Desktop)  
- Easy to navigate with clear sections (Home, About, Services, Contact)  
- Interactive with icons and social links integration  

### Key Features
- **Responsive Design:** Works perfectly on all screen sizes.  
- **Contact Section:** Email, WhatsApp, LinkedIn, GitHub links integrated with Lucide Icons.  
- **Interactive UI Elements:** Buttons, hover effects, and smooth transitions.  
- **Dark/Light Mode Support:** Seamless toggle for user preference.
